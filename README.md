# McDonald-s-Regional-Sales-Analytics(Interactive Dashboard Creation using ms Excel)
An interactive sales analytics dashboard designed to evaluate McDonald’s sales performance across South America through revenue analysis, regional insights, and KPI-driven visualizations using Excel.

## Project Objective
Analyzed McDonald’s sales performance across South America using interactive Excel dashboards to identify revenue trends, regional insights, and key business metrics.

## Questions(KPIs)
- Which region generates the highest sales revenue?
- Which product categories contribute most to overall profit?
- How do monthly sales trends vary across different regions?
- Which countries show the strongest business performance?
- What are the top-performing and low-performing products?
- How effectively is revenue growth maintained over time?
- Which regions contribute most to profitability?
- What sales patterns can be identified from customer purchasing trends?


##  Tools Used
 - Microsoft Excel
 - Data Visualization
 - KPI Reporting
 - Power Query
 - Relational Modeling (Star Schema)

## Process

- Imported and analyzed the cleaned sales dataset in Excel.
- Created KPI-driven reports using pivot tables and metrics.
- Built interactive dashboards to visualize sales and regional performance insights.
